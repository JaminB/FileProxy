Windows Explorer Client

A native Windows file browser for FileProxy. Browse connections, upload and download files, and track upload progress — all in a familiar Explorer-style interface. No runtime required — single static .exe with no dependencies.

Download
Download 64-bit (.exe) Download 32-bit (.exe)

64-bit recommended for Windows 7 and later. 32-bit for older or 32-bit-only Windows.

  1. 1 Create an API key

    The client authenticates to FileProxy using an API key. To create one:

    1. Click your username in the top-right corner and go to Profile.
    2. Open the API Keys tab and click Create.
    3. Copy the token — it is shown only once.
    Save the token somewhere safe now — you won't be able to view it again.

  2. 2 Run the client

    Double-click the downloaded .exe. On first launch a settings dialog appears. Enter your Server URL (e.g. https://fileproxy.example.com) and API Key, then click Save.

    Your credentials are saved to %APPDATA%\FileProxyExplorer\config.json so you only need to enter them once.

  3. 3 Browse your connections

    The left pane shows all your FileProxy connections. Click a connection to list its files in the right pane. Double-click a folder to navigate into it, or double-click a file to download and open it with the default application.

    Use the address bar to navigate directly to a path: 

    my-s3-connection/folder/subfolder

  4. 4 Upload, download, and delete

    • Upload — click ⬆ Upload in the toolbar to pick a local file and upload it to the current folder.
    • Download — select a file and click ↓ Download to save it locally.
    • Delete — select a file and click ✕ Delete. A confirmation is required.
    • Open — double-click any file to download it to a temp folder and open it with the default application.

  5. 5 Upload queue progress

    Large files are queued on the server for background upload. The Transfers panel at the bottom of the window shows real-time progress for all active uploads and downloads. Each transfer is represented as a progress bar with its current status (pending, uploading, or done).
